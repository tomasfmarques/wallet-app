# 💸 CAVEATS — moved

This file was split into per-module decision logs to keep Claude sessions lean
(the old 1,350-line version loaded ~65 KB into every session).

**New location:** [`docs/decisions/`](docs/decisions/) — one file per module, plus
an index at [`docs/decisions/README.md`](docs/decisions/README.md).

The full original is preserved verbatim at
[`docs/archive/CAVEATS-full.md`](docs/archive/CAVEATS-full.md).

To log a new decision, use the `/caveat` command or append to the relevant file
in `docs/decisions/`.
